## Instalación

### Kali Linux / Ubuntu / Parrot OS

```bash
git clone https://github.com/redsocietydev/nebulaurl.git
```

```bash
cd nebulaurl
````

```bash
sudo apt update && sudo apt install python3 python3-pip -y
```

```bash
pip3 install pyshorteners
```

```bash
chmod +x nebulaurl.py
```

```bash
./nebulaurl.py
```
