### **NebulaURL es una herramienta de código abierto diseñada para analizar la estructura de enlaces y comprender las técnicas de enmascaramiento mediante simulaciones de ingeniería social.**

## Probado en :

* Kali Linux
* Ubuntu
* Parrot OS

## Instalación

### Kali Linux / Ubuntu / Parrot OS

```bash
# 1. Clonar el repositorio y acceder a la carpeta
git clone https://github.com/redsocietydev/nebulaurl.git
cd nebulaurl
```

```bash
# 2. Crear el entorno virtual
python3 -m venv venv

# 3. Instalar librerías en el entorno
./venv/bin/pip install -r requirements.txt

```

```bash
# 4. Asignar permisos de ejecución al script
chmod +x nebulaurl.py
```

```bash
# 5. Ejecuta el script
./nebulaurl.py
```
